# Grace Gameonai Portfolio

A modern, dark-themed developer portfolio for Grace Gameonai, built with React and Tailwind CSS.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Structure

-   `src/components/sections`: Individual page sections (Hero, Services, Projects, etc.)
-   `src/components/layout`: Layout components (Navbar, Footer)
-   `tailwind.config.js`: Custom theme configuration (colors, fonts)

## Customization

To change colors, edit the `colors` object in `tailwind.config.js`.
To update content, edit the respective JSX files in `src/components/sections`.
